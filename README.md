# Pierre Silva Advocacia - Landing Page Premium

Site institucional desenvolvido para o escritório de advocacia trabalhista Pierre Silva.

## 🎨 Identidade Visual

- **Paleta:** Preto profundo (#0D0D0D) · Dourado/cobre (#C8852A) · Dourado claro (#E8A93C) · Off-white (#F5F0EA) · Cinza médio (#6B6B6B)
- **Tipografia:** Playfair Display (títulos) + DM Sans (corpo) + Cinzel (detalhes)
- **Estilo:** Luxury branding sóbrio, editorial escuro com acento dourado

## 🚀 Tecnologias

- **HTML5 Semântico** - Estrutura acessível e SEO-friendly
- **CSS3** - Flexbox, Grid, Variáveis CSS, Animações
- **JavaScript ES6+** - Vanilla JS, IntersectionObserver, Classes
- **Zero Frameworks** - Código 100% nativo

## 📁 Estrutura

```
├── index.html      # Estrutura completa do site
├── style.css       # Estilos com variáveis CSS e animações
```
